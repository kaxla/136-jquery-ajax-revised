# Ajaxified To-Do List

By Kayla Morrison (http://kaylahowser.com).

## Description
A to-do list that uses Ajax to perform all functions without reloading the page

## Installation

1. clone
2. run rake db:setup
3. run bundle install
4. make all the lists
